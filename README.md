# Storyteller

**Storyteller** is a 3D narrative experience set in a generated world, allowing players to interact with NPCs and engage with stories created dynamically using the Gemini API and prompt engineering techniques.

## Play the Game

You can download and play the latest build here:  
[**Oceanbound**](https://github.com/Requlish/Storyteller-Project/releases/latest)

### How to Play
1. Download the `Storyteller.zip` file from the release page.  
2. Extract all contents to a folder.  
3. Run `TDP3.exe`.

> This build is for **portfolio and demonstration purposes only**.
